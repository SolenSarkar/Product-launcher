# Product Launcher - LaunchAI Landing Page

A modern, responsive landing page for LaunchAI, built with pure HTML, CSS (custom Tailwind-like utilities), and vanilla JavaScript.

## Features
- Dark/Light theme toggle
- Smooth scrolling navigation
- Mobile-responsive dropdown menus
- Glassmorphism design
- Feature cards, testimonials, CTA sections

## Local Development
Open `landing-page/index.html` in browser or serve with:
```bash
npx serve .
```

## Deployment
Configured for GitHub Pages at `/docs` folder.

Live site: https://solensarkar.github.io/Product-launcher/
